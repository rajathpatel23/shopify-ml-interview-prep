# Shopify ML Engineer - Pair Programming Practice

Practice problems and solutions for Shopify Senior ML Engineer pair programming interview.

## Structure

```
├── problem_1_feature_pipeline/     # Feature engineering pipeline
├── problem_2_prediction_service/   # Model prediction service
├── problem_3_data_processing/      # Data processing and aggregation
├── problem_4_recommendations/      # Product recommendation system
├── problem_5_ab_testing/          # A/B test statistical analysis
├── problem_6_api_integration/     # API with caching and rate limiting
└── requirements.txt               # Python dependencies
```

## Setup

```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Interview Tips

1. **Think out loud** - Narrate your thought process
2. **Ask clarifying questions** - Understand requirements first
3. **Use AI tools** - Shopify encourages ChatGPT, Cursor, Copilot
4. **Handle edge cases** - null values, empty data, invalid inputs
5. **Write clean code** - type hints, docstrings, meaningful names
6. **Test as you go** - Walk through examples
7. **Communicate if stuck** - Discuss alternatives

## Key Phrases

- "Let me make sure I understand the requirements..."
- "I'm thinking about edge cases like..."
- "The trade-off here is..."
- "I'm going to use [AI tool] to help with..."
- "What do you think about this approach?"
